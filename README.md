
# 📝 Notes API (Backend Project)

## 📌 Overview

This is a RESTful API built using Node.js and Express.js.
It allows users to perform CRUD operations (Create, Read, Update, Delete) on notes.

---

## 🚀 Features

* Create a new note (POST)
* Get all notes (GET)
* Update a note (PATCH)
* Delete a note (DELETE)
* JSON-based request/response handling

---

## 🛠️ Tech Stack

* Node.js
* Express.js
* Postman (for API testing)

## ⚙️ Installation & Setup

1. Clone the repository:
   git clone https://github.com/asaad-codes/notes-api.git

2. Install dependencies:
   npm install

3. Run the server:
   node server.js

Server runs on: http://localhost:3000

---

## 📡 API Endpoints

GET /notes
POST /notes
PATCH /notes/:id
DELETE /notes/:id
